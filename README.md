# 📊 Social Media Engagement Dashboard
## Codec Technology Internship Project

---

## 📌 Project Overview

This project was developed as part of the **Codec Technology Internship Program**. It demonstrates the ability to collect, analyze, and visualize social media engagement data using the **YouTube Data API v3** and **Power BI**.

The dashboard tracks key engagement metrics like **likes, comments, views, and engagement rate**, and provides **data-driven recommendations** for optimizing content strategy and posting schedules.
